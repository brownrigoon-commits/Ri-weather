# 골프라이프 백엔드 설치 안내 (약 5분, 무료)

이걸 설치하면 두 가지가 켜집니다:
1. **관리자 통계** — 접속수·인기 골프장·기능 사용·연령/성별 분포 (좌표는 절대 수집 안 함)
2. **맛집 실제 사진** — 카카오맵에 등록된 '그 가게' 사진이 앱 안에 바로 표시

## 설치 순서 (사장님이 직접, 구글 계정 필요)

1. https://sheets.new 접속 → 새 스프레드시트가 열림 → 이름을 `골프라이프 통계`로 변경
2. 메뉴 **확장 프로그램 → Apps Script** 클릭
3. 열린 편집기에서 기본 코드를 모두 지우고,
   `tools/apps_script/Code.gs` 파일 내용을 **전체 복사해 붙여넣기**
4. 코드 맨 위 `ADMIN_PW = "golf2026!"` 를 **원하는 비밀번호로 변경** → 저장(💾)
5. 오른쪽 위 **배포 → 새 배포** 클릭
   - 유형 선택(⚙️): **웹 앱**
   - 설명: 아무거나 (예: v1)
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자** ← 중요!
   - **배포** 클릭 → 권한 요청이 뜨면 **허용** (본인 계정이라 안전)
6. 나오는 **웹 앱 URL** (`https://script.google.com/macros/s/…/exec`)을 복사해서
   **클로드에게 전달** — 앱에 연결해 드립니다

## 이후 관리

- 통계 원본: 방금 만든 스프레드시트의 `log` 시트에 자동 기록
- 관리자 화면: 앱 연결 후 안내드릴 숨은 주소 + 위에서 정한 비밀번호
- 코드를 수정했을 때: 배포 → 배포 관리 → ✏️ → 버전 '새 버전' → 배포
  (URL은 그대로 유지됩니다)

## 수집 항목 (개인정보 최소화)

| 수집 | 안 함 |
|---|---|
| 접속 수, 조회 골프장 이름 | ❌ 위치 좌표 (신고 의무 발생 — 절대 금지) |
| 사용 기능, 기기 종류(iOS/Android/PC), 앱 버전 | ❌ 이름·전화번호·이메일 |
| 연령대·성별 (동의자만) | ❌ 광고 식별자 |
| 무작위 기기 ID (개인 연결 불가, 순방문자 집계용) | |
